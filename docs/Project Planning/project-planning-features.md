# Overview of Holori's Project Planning key features

## Start from scratch

With holori Project planning software you can visually design your diagram using cloud vendors’ official and latest icons, customizable components and connectors. 

## Iterate on an existing infra

If you already connected a cloud account to Holori (AWS, GCP or Azure), when you open it, click on "Edit" on the top menu bar. A copy of the diagram is created on your account, it is now uncyschronized and will only evolve with the modification you make. 
Select from the list of providers icons, drag&drop them on the grid and start planning how your future infra will look like.


## Start from a template

If you don't want to start from scratch and need a bit of guidance, use our template library and start iterating from one of our predesigned diagrams. You can also save your own diagrams as templates which is the ideal solution if you often reuse the same models or want to share good practices within your organization.
