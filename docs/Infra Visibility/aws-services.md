# Covered Services AWS


:::warning

These are the covered services for Holori **infra visibility** solution, to generate **cloud diagrams**.

:::

Here are the main services supported when importing from AWS in Holori software:

- API Gateway
- AppRunner
- Athena
- Autoscaling
- Batch
- CloudFront
- DataSync
- DynamoDB
- EC2
- ECR
- ECS
- EFS
- EKS
- Elastic Beanstalk
- ELB
- FSx
- Lambda
- Lightsail
- MQ
- RDS
- S3
- SNS
- SQS

:::info

Anything missing? Let us know what you would like to see added to this list by sending us an email to support(at)holori.com
We will be happy to consider your requests.

:::
