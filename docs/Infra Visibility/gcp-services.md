:::warning

These are the covered services for Holori **infra visibility** solution, to generate **cloud diagrams**.

:::

- Automl
- Bare Metal Solution
- Batch
- Bigquery
- Bigtable
- Cloud Run
- Cloud Scheduler
- Cloud Spanner
- Cloud Storage
- Cloud Tasks
- Cloud Tpu
- Compute
- Dataproc
- Datastore
- Datastream
- Dns
- Document ai
- Filestore

- Functions
- Game Servers
- Memorystore

- Workflows
