# Navigating through the software

Navigating through Holori app is easy thanks to the left menu bar that remains visible on all pages.


![Screenshot](../images/holori-3doc25.png)

## Breadcrumb navigation

On some pages, especially once you enter into more details about an element, a navigation menu helps you understand where you are located.
If you need to move back to a previous page, simply click on the section you want to move back to.
